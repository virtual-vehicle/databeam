WebInterface

[Link:6010:Plotter Webinterface @ port 6010]

Description

Webinterface for plotting of Data.Beam live data. If you change the Port in the config make sure to adjust it in the URL as the link above uses 6010 as default port.

Configuration

  • Port: Select port for server. Must be an even number within range [6010, 6038]. Port used for websocket connection is Port + 1.
