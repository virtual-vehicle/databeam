Description

Subscribe to any integer or float values and calculate a mathematical expression.
Subscribed values are available as variables with configurable names.
Expression is evaluated when new values are received from any subscription.

Example: "sqrt(x) * (1.23 + v ** 2)"

Configuration

  • Inputs: Add subscriber channels in format module/sub_mode#channel
  • Internal Input Names: Declare a variable name per input.
  • Calculation: A mathematical expression. Can contain variables, operators and functions of python math lib.
  • Result Channel Name: The result will be published and saved under this key.

Known Issues

Calculation expression is not validated and will fail silently (exception). Expression is sanitized before being evaluated.
