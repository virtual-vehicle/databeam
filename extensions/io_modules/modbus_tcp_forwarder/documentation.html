Description

Forward configured channels as a Modbus TCP server.

Configuration

  • TCP Port: Port to listen on
  • Inputs: Add subscriber channels in format module/topic/sub_mode#channel=register.
    sub_mode may be "liveall" or "livedec".
    Registers are integers for 16-bit values - mind that 32-bit values also occupy the next register.
  • Input DBIDs: Optional list of DBIDs when connecting to external DataBeam instances

Known Issues

None.
