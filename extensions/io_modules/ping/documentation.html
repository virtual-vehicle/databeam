Description

The Ping module performs ping requests to a given list of IP addresses in a given time interval.
The round-trip time for each ping request is stored in the database.

Configuration

  • Interval [ms]: The interval between ping requests in milliseconds.
  • Timeout [ms]: The timeout for a ping request in case there is no response.
  • Servers: A list of domain names to ping, e.g., "google.at".
