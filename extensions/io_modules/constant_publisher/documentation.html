Description

The module can be used to publish configurable constant values to other modules.
A use case would be to set up values to be used with the math module.

Other uses may include simulation of other modules for debugging purposes, etc.

Configuration

  • Strings [str]: a list of strings to be published
  • Floats [double]: a list of floats to be published
  • Integers [int]: a list of integers to be published
  • Sleep Seconds [s]: decimal number of seconds to sleep between sends
