Description

Manage automatic starting and restarting of capturing.

There are two modes available, which can be combined:

  • Automatic Start: automatically start capturing after startup
  • Automatic Restart: automatically restart capturing (e.g. to reduce file sizes)

Configuration

  • Capture On Start Enabled: automatically start capturing after startup
  • Capture On Start Delay S [s]: delay in seconds to wait after startup
  • Capture Restart Enabled: automatically restart capturing
  • Capture Restart Time [hh:mm:ss]: when to perform restart
  • Capture Restart Interval [day/hour/minute]:
    When "day" is selected, restart at specified time.
    When "hour" is selected, only "mm:ss" is considered.
    When "minute" is selected, only "ss" is considered.
