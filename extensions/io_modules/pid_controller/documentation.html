Description

PID controller with live-data input and output for real-time calculation. Uses velocity algorithm.

Output:

  • "output": controller output value
  • "p_share": proportional part of output
  • "i_share": integral part of output
  • "d_share": derivative part of output
  • "pv": used input process value for this sample

Configuration

  • Force On: set output to maximum
  • K P: P-gain
  • K I: I-gain
  • K D: D-gain
  • Period S [s]: how often the controller is updated
  • Output Max: maximum output value
  • Output Min: minimum output value
  • Inverted: either raise or lower output if input is smaller setpoint
  • Setpoint: setpoint of controller
  • Input Module: which module to subscribe to
  • Input Channel: input channel
  • Input Sub Mode [liveall/livedec]: input subscription mode
  • Input Valid Min: Minimum valid input value.
    Controller will set output to minimum if exceeded.
  • Input Valid Max: Maximum valid input value.
    Controller will set output to minimum if exceeded.
