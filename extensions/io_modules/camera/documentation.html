Description

This module captures video frames from cameras that support UVC (USB Video Class).

Configuration

  • Resolution: Capture resolution for the camera.
  • FPS: Number of video frames per second captured from the camera.
  • Codec: Codec for video capture.
  • Rotation: Rotation of the frame (e.g. to correct camera orientation).
  • Dot In Center Size: Size of red dot in the center (0 to disable).
  • Crosshair Size: Size of red crosshair (0 to disable).
  • Enable Mp4: Enables writing frames to mp4 file.
  • Mp4 Interval [s]: Interval in seconds to capture frames to mp4 file. A value of 0 writes every frame as given by FPS.
  • Enable Mcap: Enables writing frames to mcap file.
  • Mcap Resolution: Resolution to use for mcap frames. This should not exceed Resolution.
  • Mcap Interval [s]: Interval in seconds to capture frames to mcap file. A value of 0 writes every frame as given by FPS.
  • Enable Preview: Enables writing frames for preview display.
  • Preview Resolution: Resolution to use for preview frames. This should not exceed Resolution
  • Live Source: Select source for live data forwarding.

To find supported Resolution, FPS and Codec of the camera use
"v4l2-ctl -d /dev/video0 --list-formats-ext"
where /dev/video0 should match the video index of the connected camera.
