Description

Parse NMEA sentences from GNSS devices via serial port or TCP stream.

Supported messages: "GGA" (for position and meta-data), "THS" (for heading), "VTG" (for speed)

Configuration

  • Input Mode: serial or TCP/IP
  • Hostname: hostname used for TCP/IP connection
  • Tcp Port: port used for TCP/IP connection
  • Serial Port: serial device used for serial connection
  • Serial Baudrate: baud rate used for serial connection
  • Ntrip Client: use NTRIP client to send correction data to module
  • Ntrip Vrs Gga Sending Interval S: -1 = OFF : how often to send GGA position to NTRIP caster
  • Ntrip Server: hostname of server
  • Ntrip Port: port number
  • Ntrip Mountpoint: station identifier
  • Ntrip User: auth credentials
  • Ntrip Password: auth credentials
  • Ntrip Timeout: if no data is received for X seconds, restart client

Known Issues

None.
