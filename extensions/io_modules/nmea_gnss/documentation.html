Description

Parse NMEA sentences from GNSS devices via serial port or TCP stream.

Supported messages: "GGA" (for position and meta-data), "THS" (for heading), "VTG" (for speed)

Configuration

  • Input Mode: serial or TCP/IP
  • Hostname: hostname used for TCP/IP connection
  • Tcp Port: port used for TCP/IP connection
  • Serial Port: serial device used for serial connection
  • Serial Baudrate: baud rate used for serial connection

Known Issues

None.
