Description

A module to receive data over a UDP connection. Allows for bytestream or json formatted data.

Configuration

  • UDP Address: The UDP address to listen to.
  • UDP Port: The UDP port to use with the address.
  • Use Length Bytes: Toggle this, if the data sender sends the length of each data packet before the data itself. Allows for more robust data transfer.
  • Number Length Bytes: If a length byte is used, this field defines how many bytes the length uses.
  • Big Endian: If set, all data and the length is interpreted in big-endian.
  • Data Format: Defines the type of data which is sent to the module.
    • struct: Expects a byte array. Please use the "Struct Entries" field to define what type of data is expected.
    • json: Expects a json-formatted data packet. This setting ignores the "Struct Entries" field.
  • Struct Entries: If "Data format" is set to struct, this field defines the data which is expected.
