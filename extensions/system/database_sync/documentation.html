Description

Synchronize data from MCAP measurements to TimescaleDB database.
For each module topic, a table will be created in the database.

Configuration

  • Start Sync: manually start synchronization
  • Stop Sync: stop currently running synchronization
  • Force Re-Sync: try to upload data with all timestamps (not only the latest)
  • Sync On Capture Stop: automatically sync when capture is stopped
  • Delay before start [s]: wait a little before starting the synchronization
  • Db Host Port: host:port of the timescaledb instance
  • Db User: timescaledb username
  • Db Password: timescaledb password
  • Db Name: timescaledb database name
  • Sync Modules: list of module names to sync

Known Issues

Currently only floating-point values are supported.
