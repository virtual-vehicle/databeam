Description

A system module to allow remote browsing of the current DataBeam project data directory.

Configuration

  • Port: The port the filebrowser is running on.
  • Username: The username needed for filebrowser access.
  • Password: The password needed for filebrowser access.
  • Allow Delete Data: If ticked, the user can delete files and directories. Use at your own risk, as files and directories are not recoverable when deleted.
  • Use Single Click: If ticked, the filebrowser will use single click instead of double click. (requires Docker restart)

Known Issues

None.
